# Proyecto: Contador de 0 a 99 con Display 7 Segmentos y Multiplexación
![image](https://github.com/fedeveemo/SPD-Multiplexacion-Parcial1/assets/146194035/a8e7ed93-f896-4635-b92b-d4b4addf1f57)

# Función Principal
- La funcion acerca de este arduino es la de mostrar numeros del 0 al 99 mediante 2 display de manera que se use la multiplexacion, con 3 botones que permiten controlar los display sumando, restando y resenteando a 0 el contador.

# Proyecto: Switch Numeros Primos
![image](https://github.com/fedeveemo/SPD-Multiplexacion-Parcial1/assets/146194035/609e3c03-330b-4837-b6a6-20ec4e840139)

# Función Principal
- La funcion sobre esta parte del trabajo es un contador con 2 display como el primer arduino, pero con la diferencia de que tiene un switch agregado, que hace que se intercale la forma de mostrar los numeros. Se mostraran los numeros como en el primer trabajo, pero tambien tendra la opcion de solo mostrar en el display los numeros primos.
